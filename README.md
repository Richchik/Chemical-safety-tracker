# Chemical Safety Tracker

A web-based dashboard to track chemical safety incidents in real time.

## Features
- Incident listing with severity levels
- Pie chart visualization of risk levels
- Safety awareness videos linked to incidents
- Real-time updates using Firebase

## Technologies Used
- HTML, CSS, JavaScript
- Firebase Realtime Database
- Firebase Hosting
- Google Forms & Google Sheets
